# 临时文件目录

此目录用于存放测试运行时生成的临时文件。

## 说明

- 此目录中的所有内容都会被自动清理
- 不要在此目录中放置重要文件
- 运行 `python test/scripts/cleanup_temp_files.py` 可手动清理

## 自动清理

测试框架会在以下时机自动清理临时文件：
1. 测试会话开始前
2. 每个测试函数结束后
3. 测试会话结束后

## 目录结构

```
temp/
├── temp_<pid>/          # 进程级临时目录
└── temp_*               # 其他临时文件
```

> **注意**: 此目录内容已在 `.gitignore` 中，不会被提交到版本控制。
